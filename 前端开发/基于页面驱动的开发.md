# 基于页面驱动的开发



####最开始，只有一个简单的页面A.html
```html
<!-- A.html大致如下 -->

<link href="a.css" />
...
<script src="a.js"></script>
```
####后来,又增加了一个页面B.html,同时,A.html和B.html有公共的部分,于是

```html
B.html

<link href="a_b.css" />
<link href="b.css" />
<script src="a_b.js"></script>
<script src="b.js"></script>
```

```html
<!-- A.html变成了下面这样 -->
<!-- 当然此时a.css及a.js已不是 -->

<link href="a_b.css" />
<link href="a.css" />
<script src="a_b.js"></script>
<script src="a.js"></script>
```
