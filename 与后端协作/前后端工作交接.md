# 前后端工作交接
* 前端需要把http页面源文件或地址交付给研发
* 告知后端开发人员入口脚本为id="atmjsuse"的script标签内的atmjs.use处

```js
// 注意看script标签的id
// 
<script id="atmjsuse" data-from="atmjs" type="text/javascript">
    atmjs.use("m/mis:mis");    // 'm/mis:mis' 即为入口脚本
</script>
```
