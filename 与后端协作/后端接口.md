* 生成js标签

```php

```
 
* 生成回调函数

```php
// php demo
 <?php Atmjs.setJs("function (exp) {
console.log(exp);
}"); ?>
// output
<script type="text/javascript">
atmjs.use('project/module:file', function (exp) {
  console.log(exp);
});
</script>
```
