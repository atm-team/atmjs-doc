# 后端接口

* 声明入口文件

```php
// php demo
 <?php Atmjs.use('project/module:file'); ?>
```
 
* 生成css标签

```php
// php demo
 <?php Atmjs.loadCss(); ?>
```
 
* 生成js标签

```php
// php demo
 <?php Atmjs.loadJs(); ?>
```
 
* 生成回调函数

```php
// php demo
 <?php Atmjs.setJs(); ?>
// output
<script>
atmjs.use('project/module:file', function (exp) {
  console.log('exp')
});
</script>
```
